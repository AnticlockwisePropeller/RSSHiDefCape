RSSHiDefCape
AnticlockwisePropeller
GitHub: https://github.com/AnticlockwisePropeller/RSSHiDefCape
This mod adds a high(er)-definition heightmap to Cape Canaveral and the surrounding area in Real Solar System.

Many thanks to everyone in the RO/RSS GitHub for generally being an awesome bunch of people!

License: CC-BY-NC-SA
Terrain artwork by AnticlockwisePropeller
Heightmap derived from U.S. Geological Survey data, which is considered to be in the Public Domain
https://www.usgs.gov/information-policies-and-instructions/copyrights-and-credits

Dependencies:
- Real Solar System v16.4 for KSP v1.7.3 (and all of its dependencies): See GitHub for details
https://github.com/KSP-RO/RealSolarSystem
